# 🐍 Simple Python Project Suite

Collection de **5 projets Python utiles** pour automatiser des tâches quotidiennes.

## 📂 Structure du projet

```
Simple_python_project/
├── ytDownloader/          # Télécharger vidéos YouTube
├── imageProcessor/        # Éditer des images
├── pdf/                   # Fusionner des PDF
├── ocr/                   # Reconnaissance de texte (OCR)
├── fileOrganizer/         # Organiser les fichiers
├── watermarking/          # Ajouter des filigranes
├── duplicateFinder/       # Trouver les fichiers en doublon
├── passwordManager/       # Gérer les mots de passe
├── csvAnalyzer/          # Analyser les données CSV
└── requirements.txt       # Dépendances
```

## 🚀 Démarrage rapide

### Installation des dépendances
```bash
pip install -r requirements.txt
```

### Dépendances principales
- **Pillow** - Traitement d'images
- **PyPDF2** - Manipulation de PDF
- **pytube** - Téléchargement YouTube
- **easyocr** - Reconnaissance optique de caractères
- **opencv-python** - Traitement vidéo/images
- **pandas** - Analyse de données

## 📝 Notes importantes

- Testez toujours en mode **simulation** avant d'appliquer les changements
- Faites des **sauvegardes** avant de supprimer des fichiers
- Les fichiers du Password Manager doivent être **sécurisés**
- L'OCR fonctionne mieux avec du texte **imprimé** qu'avec du manuscrit
